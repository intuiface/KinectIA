Pré-requis pour la compilation du projet :
	- Installation de la SDK kinect 1.7 pour Microsoft (http://www.microsoft.com/en-us/kinectforwindows/develop/developer-downloads.aspx)
	- Installation de la SDK Microsoft Speech Platform 11 (http://www.microsoft.com/en-us/download/details.aspx?id=27226)

Pré-requis pour utiliser la dll :
	- Installation du Runtime 1.7 du Kinect pour Microsoft (http://www.microsoft.com/en-us/kinectforwindows/develop/developer-downloads.aspx)(rubrique Related Download)
	- Installation du Runtime Microsoft Spee Platform 11 si vous utilisez la reconnaissance vocale (http://www.microsoft.com/en-us/download/details.aspx?id=27225)
